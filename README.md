# Bank-Marketing-Campaign-Analysis-using-Python


## Project Overview

This project analyzes a bank marketing dataset to identify factors influencing customer subscriptions to term deposits.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights

- Students had the highest subscription rate.
- Retired and unemployed customers also showed high subscription rates.
- Call duration had the strongest relationship with subscriptions.
- Most customers were married and worked in administrative roles.
